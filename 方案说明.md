# 「未来鹅」大学生职业成长 AI 陪伴平台 - 方案说明

## 一、问题诊断

### 1.1 核心痛点
- **信息触达单向化**：企业校招信息主要通过公众号推文、宣讲会等单向渠道传播，缺乏互动反馈机制
- **年级断层严重**：现有资源主要面向大四等"已有明确求职意向"的高年级学生，缺少对中低年级同学的早期连接
- **个性化缺失**：缺少"一对一陪伴"的体验和情感链接，无法针对每个学生的不同阶段和需求提供差异化指导
- **认知门槛高**：大一、大二学生对互联网行业和腾讯缺乏系统的认知入口，难以建立早期兴趣

### 1.2 用户分层需求
| 阶段 | 核心困惑 | 陪伴需求 |
|------|----------|----------|
| 大一 | 互联网行业是什么？腾讯是做什么的？ | 行业认知、兴趣探索 |
| 大二 | 专业和职业如何关联？ | 方向选择、技能规划 |
| 大三 | 鹅厂有什么岗位？如何准备？ | 实习信息、面试准备 |
| 大四 | 秋招如何应对？offer怎么选？ | 求职实战、决策支持 |
| 研究生 | 专业优势如何转化？ | 高端岗位、学术转化 |

## 二、方案设计

### 2.1 产品定位
「未来鹅」是一款面向在校大学生的 **AI 驱动型职业成长陪伴平台**，通过智能化对话、个性化规划和精准资源推荐，将"单向招聘"转化为"持续成长陪伴"，帮助企业在学生进入正式求职窗口前建立早期连接。

### 2.2 核心功能架构
1. **年级身份选择**：用户首次访问选择年级（大一至研究生），系统自动匹配对应阶段的陪伴内容
2. **AI 智能对话**：提供三种对话场景（职业咨询/模拟面试/简历诊断），基于关键词+规则引擎的模拟 AI 回复，提供真实对话体验
3. **个人画像测评**：5 道快速测评题生成能力雷达图，展示用户优势维度
4. **成长路径规划**：根据年级展示时间轴式成长里程碑和配套资源推荐
5. **鹅厂专区**：集中展示腾讯热门岗位、企业文化价值观和校招培养体系

### 2.3 用户体验流程
```
访问首页 → 选择年级 → 完成测评 → AI对话陪伴 → 查看成长规划 → 浏览鹅厂专区
```

## 三、AI 工具选型与配置

### 3.1 技术方案
| 层级 | 技术选型 | 选型理由 |
|------|----------|----------|
| 前端框架 | React 18 + TypeScript | 组件化开发、类型安全、生态丰富 |
| 构建工具 | Vite 5 | 极速冷启动、HMR 热更新 |
| 样式方案 | TailwindCSS 3 | 原子化 CSS、响应式设计支持 |
| 状态管理 | Zustand 5 | 轻量级、API 简洁、无样板代码 |
| 路由系统 | React Router 7 | React 生态标准路由 |
| 图表库 | Chart.js + react-chartjs-2 | 轻量、API 友好、雷达图支持 |
| 图标库 | Lucide React | 现代化线性图标、体积小 |

### 3.2 AI 对话实现方案
由于是 Demo 环境，采用 **关键词匹配 + 规则引擎** 的模拟 AI 对话：
- **分级响应**：根据不同年级返回对应阶段的针对性建议（每个年级预设 4 条专业回复）
- **场景切换**：职业咨询、模拟面试、简历诊断三种场景切换回复策略
- **关键词识别**：识别"腾讯""简历""面试""算法"等高频关键词，返回精准回答
- **快捷问题**：为每个年级预设 3 个高频问题，降低交互门槛
- **打字效果**：模拟 1-2 秒的 AI "思考"延迟和打字动画，提供真实对话体验

### 3.3 关键配置
- **数据持久化**：使用 LocalStorage 保存用户画像和对话历史（Demo 版本）
- **Mock 数据**：预置成长路径、岗位信息、文化价值观等静态数据
- **响应式设计**：桌面端优先，兼容平板和移动端

## 五、效果评估

### 5.1 用户体验指标
- **页面加载速度**：首屏加载 < 2 秒（构建后 JS 138KB gzip）
- **交互响应**：对话 AI 回复延迟 1-2 秒，打字效果流畅
- **移动端适配**：完美适配 375px-1440px 屏幕宽度

### 5.2 核心价值验证
- **早期连接**：通过年级分层，覆盖从大一至研究生的全周期用户
- **情感链接**：AI 对话陪伴提供类"一对一咨询"的体验
- **雇主品牌**：鹅厂专区展示企业文化和培养体系，传递雇主品牌价值

## 六、部署说明

```bash

npm install

# 启动开发服务器
npm run dev

http://localhost:5173/
```

```bash

npm run build

# 部署 dist 文件夹到以下任一平台：
```
