## 📁 项目概览

### 产品定位
**未来鹅** 是一款面向在校大学生（大一至研究生）的 AI 职业成长陪伴平台，通过智能化对话、个性化规划和精准资源推荐，将"单向招聘"转化为"持续成长陪伴"。

### 核心价值主张
- **AI驱动**：接入智谱 GLM 系列模型，支持流式响应、思维链展示
- **腾讯生态**：深度融入腾讯六大事业群信息、校招流程、企业文化
- **陪伴成长**：从大一到大四的全周期职业规划与成长路径跟踪
- **品牌升级**：2026年5月启动视觉与体验升级，引入腾讯TDesign色彩系统

### 核心功能模块

| 功能模块 | 描述 | 状态 | 质量评级 |
|---------|------|------|---------|
| 首页 | 品牌展示、年级选择、功能导航、"自树而生"主题 | ✅ 升级完成 | ⭐⭐⭐⭐⭐ |
| AI 对话 | 职业咨询/模拟面试/简历诊断 3 场景 + 流式输出 | ✅ 完成 | ⭐⭐⭐⭐⭐ |

## 🏗️ 代码架构审计

### 架构分层分析

```
应用层 (App.tsx)
  ├── 路由层 (React Router v7)
  │     ├── 页面组件 (lazy + Suspense)
  │     └── 全局组件 (Header, Toast, Onboarding)
  ├── 组件层
  │     ├── UI 基础组件 (Button, Card, Modal, Badge, Input)
  │     ├── 业务组件 (ChatInterface, RadarChart, GrowthTimeline)
  │     └── 工具组件 (ErrorBoundary, PageSkeletons, MarkdownRenderer)
  ├── 状态层 (Zustand)
  │     ├── 主状态 (appStore) - 用户数据、对话、收藏
  │     ├── 辅助状态 (toastStore, performanceStore)
  │     └── 持久化配置 (persist middleware)
  ├── 服务层
  │     ├── AI API (aiApi.ts) - 流式、错误处理、Prompt 构建
  │     └── 腾讯 API (tencentApi.ts) - 岗位数据
  ├── 工具层
  │     ├── 错误处理 (errorHandling.ts)
  │     ├── 性能监控 (webVitals, performanceMonitor)
  │     ├── 安全存储 (secureStorage.ts)
  │     └── 数据工具 (progressCalculator, resourceFilter, export)
  └── 设计系统层
        ├── CSS 变量令牌 (index.css :root)
        ├── Tailwind 配置映射 (tailwind.config.js)
        ├── 微交互系统 (micro-interactions.css)
        └── 主题系统 (dark mode, 生长主题)
```

#### 1. 流式响应处理
```typescript
// [aiApi.ts:378-502](file:///c:/Users/MECHREVO/OneDrive/Desktop/未来鹅/src/services/aiApi.ts#L378-L502)
export async function callAIAPIStream(
  messages: ChatMessage[],
  grade: UserGrade | null,
  scene: ChatSceneId,
  onChunk: (chunk: string) => void,
  onReasoningChunk?: (chunk: string) => void
): Promise<{ content: string; reasoning_content: string }> {
  const reader = response.body.getReader();
  const decoder = new TextDecoder();
  let buffer = '';
  
  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    
    buffer += decoder.decode(value, { stream: true });
    const lines = buffer.split('\n');
    buffer = lines.pop() || '';
    
    for (const line of lines) {
      const trimmed = line.trim();
      if (!trimmed || !trimmed.startsWith('data:')) continue;
      
      const dataStr = trimmed.slice(5).trim();
      if (dataStr === '[DONE]') continue;
      
      const parsed = JSON.parse(dataStr);
      const delta = parsed.choices?.[0]?.delta;
      if (delta?.content) {
        fullContent += delta.content;
        onChunk(delta.content);
      }
    }
  }
  
  return { content: fullContent, reasoning_content: fullReasoningContent };
}
```

```typescript
// [aiApi.ts:248-316](file:///c:/Users/MECHREVO/OneDrive/Desktop/未来鹅/src/services/aiApi.ts#L248-L316)
function parseAPIError(status: number, errorData): string {
  const code = errorData?.error?.code;
  
  if (status === 401 || code === '1000') return 'API_KEY_INVALID';
  if (code === '1113') return 'ACCOUNT_BALANCE';
  if (code === '1211') return 'MODEL_NOT_FOUND';
  if (code === '1310') return 'RATE_LIMIT';
  if (status >= 500) return 'SERVER_ERROR';
  
  return 'UNKNOWN';
}
```
